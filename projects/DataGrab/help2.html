Instructions:
  1. Upload an image file via "Choose File"
  2. Choose four reference points from axes:
    • Place mouse at given point
    • Press '1' for x1, '2' for x2, '3' for y1, and '4' for y2
    • Input values for x1, x2, y1, y2 under 'Axis Definitions'
  3. Now that the scales have been set points can be added:
    • Simply click on any location on plot to add point to collection
    • Alternatively, place cursor on plot and hit 'enter'
    • Once cursor is on plot arrow keys can be used for better accuracy
    • Use '+' and '-' to adjust zoom
  4. Use 'Create text file' to export values
